# Module 2 – Mathematical Physics I: Vector and Matrix Methods

This module introduces computational implementation of vector algebra and matrix operations which are fundamental tools in physics.

Students learn to represent scalars and vectors computationally and perform mathematical operations using Python.

---

## Learning Objectives

After completing this module students will be able to:

• represent scalar and vector quantities computationally  
• perform vector algebra operations  
• compute scalar and vector products  
• evaluate vector differential operators  
• implement matrix methods using Python  

---

## Topics Covered

Scalars and vectors  

Collinear vectors  

Addition and subtraction of vectors  

Scalar product  

Vector product  

Vector triple product  

Gradient  

Divergence  

Curl  

Matrix methods  

Matrix operations  

Arrays
