# Module 1 – Fundamentals of Computation and Data Visualization

This module introduces the fundamental computational tools required for scientific programming and data analysis in physics.

Students learn how to visualize scientific data, interpret graphical information, and perform interpolation and data fitting using Python.

---

## Learning Objectives

After completing this module students will be able to:

• understand basic computational methods used in scientific computing  
• visualize scientific data in two and three dimensions  
• interpret graphical data and extract physical meaning  
• perform interpolation of discrete datasets  
• apply curve fitting methods to experimental data  

---

## Topics Covered

Fundamentals of computation  

Data visualization in 2-dimension  

Data visualization in 3-dimension  

Plotting scientific data  

Data interpretation in graphical form  

Interpolation of datasets  

Curve fitting techniques
