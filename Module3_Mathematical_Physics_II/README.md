# Module 3 – Mathematical Physics II: Numerical Methods and Statistical Analysis

This module focuses on computational techniques used for numerical analysis and statistical interpretation of data.

Students will implement algorithms for solving mathematical problems commonly encountered in physics.

---

## Learning Objectives

After completing this module students will be able to:

• generate sequential and random numbers  
• analyze statistical properties of datasets  
• perform error analysis in measurements  
• implement numerical methods for root finding  
• compute derivatives and integrals numerically  
• solve ordinary and partial differential equations
